## Get started

*Note that you will need to have [Node.js](https://nodejs.org) installed.*

Install the dependencies...
```bash
npm install
cd client
npm install
```

In backend and frontend just hit...
```bash
npm start
cd ..
npm start
```

```bash
Backend will run on port 5000
Frontend will run on http://localhost:3000
```
